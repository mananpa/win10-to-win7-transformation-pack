You need to follow some steps.
1. Install Winaero Tweaker
2. Launch it
3. If you use 1903 or higher, disable the lock screen blur (Use the search feature option)
4. Enable old volume control
5. Disable Action Center
6. Enable balloon tooltips
7. Enable startup sound - it will have the Windows 7 startup sound by default.
8. Disable cortana
9. Show BSOD, Disable smileyr
10. Get classic apps (Win photo viewer, Calculator, Games, Sticky notes, Gadgets)
11. Enable Battery flyout (If you use laptop)
12. Add personalization (Classic personalization, classic color and appearance, Classic desktop background)
This will enable some legacy leftovers from Windows 7.
